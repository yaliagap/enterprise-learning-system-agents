## Skill Area: Describe cloud concepts (25–30%)

### Q1
**Question:** A company is planning to move its workloads to Azure. The IT team wants to ensure that if one datacenter in a region goes offline, the application remains available. Which Azure concept directly addresses this requirement?

**Options:**
- A. Scalability
- B. Availability Zones
- C. Resource Groups
- D. Azure Blueprints

**Correct Answer:** B  
**Explanation:** Availability Zones are physically separate datacenters within an Azure region, each with independent power, cooling, and networking. Deploying resources across multiple Availability Zones protects against single datacenter failures and provides high availability.

---

### Q2
**Question:** What is the shared responsibility model in cloud computing?

**Options:**
- A. Microsoft is responsible for all security in Azure
- B. The customer is responsible for all security aspects of their deployment
- C. Security responsibilities are divided between the cloud provider and the customer based on the service model (IaaS, PaaS, SaaS)
- D. Security responsibility only applies to on-premises environments

**Correct Answer:** C  
**Explanation:** In the shared responsibility model, Microsoft manages physical security, hardware, and the underlying cloud infrastructure. The customer's responsibility shifts depending on the service model: IaaS customers manage OS, middleware, and apps; PaaS customers manage only apps and data; SaaS customers manage only data and access.

---

### Q3
**Question:** Which cloud service model gives customers the MOST control over the underlying infrastructure?

**Options:**
- A. Software as a Service (SaaS)
- B. Platform as a Service (PaaS)
- C. Infrastructure as a Service (IaaS)
- D. Function as a Service (FaaS)

**Correct Answer:** C  
**Explanation:** IaaS provides the highest level of control, allowing customers to manage the operating system, middleware, runtime, applications, and data. The cloud provider manages only the physical hardware, network, and virtualization layer.

---

### Q4
**Question:** A retail company experiences a 10x spike in website traffic every December. They want to add compute resources automatically during peak periods and remove them afterward to minimize costs. Which cloud benefit does this describe?

**Options:**
- A. High availability
- B. Fault tolerance
- C. Elasticity
- D. Disaster recovery

**Correct Answer:** C  
**Explanation:** Elasticity is the ability to automatically scale resources up or down in response to demand. This contrasts with scalability (the ability to scale, but not necessarily automatically) and high availability (keeping services running despite failures).

---

### Q5
**Question:** Which of the following is a characteristic of a PRIVATE cloud?

**Options:**
- A. Resources are shared among multiple organizations
- B. Unlimited on-demand scalability with zero upfront costs
- C. Dedicated infrastructure hosted exclusively for one organization
- D. Managed entirely by a third-party cloud provider

**Correct Answer:** C  
**Explanation:** A private cloud provides computing resources exclusively for a single organization. It can be hosted on-premises or in a dedicated section of a third-party datacenter. Unlike public clouds, resources are not shared with other organizations.

---

### Q6
**Question:** What is the primary advantage of the consumption-based pricing model in cloud computing?

**Options:**
- A. Fixed monthly costs regardless of usage
- B. You pay only for the resources you use, avoiding upfront capital expenditures
- C. Free access to all cloud services with a subscription fee
- D. Guaranteed price locks for three years

**Correct Answer:** B  
**Explanation:** The consumption-based (pay-as-you-go) model converts capital expenditures (CapEx) to operational expenditures (OpEx). Organizations avoid large upfront investments and pay only for what they consume, enabling cost efficiency and flexibility.

## Skill Area: Describe Azure architecture and services (35–40%)

### Q7
**Question:** A developer needs to run a containerized microservices application without managing the underlying virtual machines or orchestration infrastructure. Which Azure service is MOST appropriate?

**Options:**
- A. Azure Virtual Machines
- B. Azure Container Apps
- C. Azure Virtual Machine Scale Sets
- D. Azure Dedicated Host

**Correct Answer:** B  
**Explanation:** Azure Container Apps is a serverless container service built on Kubernetes that abstracts away infrastructure management. Developers deploy containers without configuring VMs, clusters, or orchestration. Azure Virtual Machines and Scale Sets require OS and infrastructure management.

---

### Q8
**Question:** Which Azure storage service is optimized for storing massive amounts of unstructured data such as images, videos, and log files?

**Options:**
- A. Azure Queue Storage
- B. Azure Table Storage
- C. Azure Blob Storage
- D. Azure File Storage

**Correct Answer:** C  
**Explanation:** Azure Blob Storage (Binary Large Object) is designed for storing unstructured data at scale. It supports hot, cool, and archive access tiers for cost optimization. Queue Storage is for messaging, Table Storage for NoSQL key-value data, and File Storage for SMB file shares.

---

### Q9
**Question:** An organization has virtual machines in Azure that need to communicate with on-premises servers. The network team requires a private, encrypted connection over the public internet. Which Azure networking service should they use?

**Options:**
- A. Azure ExpressRoute
- B. Azure VPN Gateway
- C. Azure Content Delivery Network
- D. Azure Load Balancer

**Correct Answer:** B  
**Explanation:** Azure VPN Gateway creates encrypted tunnels (IPsec/IKE) between Azure virtual networks and on-premises networks over the public internet. ExpressRoute provides a private, dedicated connection that does NOT traverse the public internet — it is preferred for higher bandwidth and lower latency but requires a connectivity provider.

---

### Q10
**Question:** Which Azure database service provides a fully managed, serverless relational database compatible with Microsoft SQL Server?

**Options:**
- A. Azure Cosmos DB
- B. Azure Database for MySQL
- C. Azure SQL Database
- D. Azure Database for PostgreSQL

**Correct Answer:** C  
**Explanation:** Azure SQL Database is a fully managed PaaS relational database service based on the latest stable version of Microsoft SQL Server. It supports serverless compute tiers that automatically pause and resume based on usage. Azure Cosmos DB is a NoSQL multi-model database.

---

### Q11
**Question:** What is the purpose of Azure Availability Sets?

**Options:**
- A. To distribute VMs across different Azure regions
- B. To protect VMs from hardware failures within a single datacenter by placing them in different fault and update domains
- C. To automatically scale the number of VMs based on CPU usage
- D. To enable private connectivity between Azure and on-premises networks

**Correct Answer:** B  
**Explanation:** Availability Sets protect against hardware failures and planned maintenance within a single datacenter by organizing VMs into fault domains (separate power and network switches) and update domains (groups that restart during planned maintenance). For protection against datacenter failures, use Availability Zones instead.

---

### Q12
**Question:** An application needs to distribute incoming HTTPS web traffic across multiple backend VMs in different Azure regions. Which Azure service provides this capability?

**Options:**
- A. Azure Load Balancer
- B. Azure Application Gateway
- C. Azure Traffic Manager
- D. Azure Front Door

**Correct Answer:** D  
**Explanation:** Azure Front Door is a global load balancing service that distributes traffic across regions based on latency, priority, or weighted routing. It operates at Layer 7 (HTTP/HTTPS). Azure Load Balancer operates at Layer 4 within a region. Azure Traffic Manager is DNS-based global routing. Application Gateway is a regional Layer 7 load balancer.

---

### Q13
**Question:** Which Azure Compute service allows you to run event-driven code without provisioning or managing servers, charging only for execution time?

**Options:**
- A. Azure App Service
- B. Azure Functions
- C. Azure Kubernetes Service
- D. Azure Batch

**Correct Answer:** B  
**Explanation:** Azure Functions is a serverless compute service that executes code in response to events (HTTP triggers, timer triggers, queue messages, etc.). You are billed only for the time the function executes and the number of executions, with no charge during idle periods.

## Skill Area: Describe Azure management and governance (30–35%)

### Q14
**Question:** A company wants to enforce that all Azure resources must have a specific tag (e.g., "CostCenter") applied during creation and that non-compliant resources are flagged automatically. Which Azure service accomplishes this?

**Options:**
- A. Azure Blueprints
- B. Azure Policy
- C. Azure Role-Based Access Control (RBAC)
- D. Microsoft Defender for Cloud

**Correct Answer:** B  
**Explanation:** Azure Policy evaluates resources against defined rules and enforces organizational standards. Policies can audit, deny, or auto-remediate non-compliant resources. For example, a policy can require the "CostCenter" tag on all resources and flag or block resource creation that lacks it.

---

### Q15
**Question:** What is the purpose of Management Groups in Azure?

**Options:**
- A. To group individual resources like VMs and databases for billing purposes
- B. To organize multiple Azure subscriptions into a hierarchy for applying governance policies and RBAC at scale
- C. To manage the lifecycle of Azure resource templates
- D. To control network traffic between virtual networks

**Correct Answer:** B  
**Explanation:** Management Groups sit above subscriptions in the Azure hierarchy (Management Group → Subscription → Resource Group → Resource). Azure Policy and RBAC assignments at the Management Group level are inherited by all subscriptions and resources beneath them, enabling governance at scale across an entire organization.

---

### Q16
**Question:** Which tool allows you to estimate the monthly cost of Azure services before deploying them?

**Options:**
- A. Azure Cost Management
- B. Azure TCO Calculator
- C. Azure Pricing Calculator
- D. Azure Advisor

**Correct Answer:** C  
**Explanation:** The Azure Pricing Calculator lets you configure specific Azure services and view estimated monthly costs before deployment. The TCO Calculator estimates cost savings from migrating on-premises workloads to Azure. Azure Cost Management analyzes and optimizes actual spending after resources are deployed.

---

### Q17
**Question:** A security team wants to view recommendations to improve the security posture of their Azure resources, including compute, networking, and data. Which Azure service provides this?

**Options:**
- A. Azure Monitor
- B. Microsoft Defender for Cloud
- C. Azure Sentinel
- D. Azure Key Vault

**Correct Answer:** B  
**Explanation:** Microsoft Defender for Cloud (formerly Azure Security Center) provides a unified security management system that evaluates the security configuration of Azure resources and provides a Secure Score with prioritized recommendations. Microsoft Sentinel is a SIEM/SOAR solution for threat detection and response.

---

### Q18
**Question:** What does Azure Role-Based Access Control (RBAC) allow administrators to do?

**Options:**
- A. Encrypt data at rest in Azure Storage
- B. Monitor the performance of Azure VMs
- C. Grant specific permissions to users, groups, or applications at a specific scope (management group, subscription, resource group, or resource)
- D. Automate the deployment of Azure infrastructure using templates

**Correct Answer:** C  
**Explanation:** Azure RBAC uses role assignments to control who can access Azure resources and what they can do. A role assignment combines a security principal (user, group, service principal, or managed identity), a role definition (set of permissions like Reader, Contributor, Owner), and a scope. Permissions are inherited downward through the scope hierarchy.

### Q19
**Question:** Which Azure service provides a free online assessment to renew certifications every 24 months without retaking the full proctored exam?

**Options:**
- A. Microsoft Learn Practice Assessments
- B. Microsoft Certification Renewal Assessment
- C. Azure Skills Challenge
- D. Microsoft Virtual Training Days

**Correct Answer:** B  
**Explanation:** Microsoft offers free online renewal assessments through Microsoft Learn to renew most role-based and specialty certifications. Renewals are non-proctored, free, and available in the six-month window before the certification expires. This applies to AZ-900 and most other certifications on a 24-month renewal cycle.
