## Skill Area: Describe artificial intelligence workloads and considerations (20–25%)

### Q1
**Question:** A retail company uses an AI system to predict which customers are likely to cancel their subscription in the next 30 days. This is an example of which type of machine learning task?

**Options:**
- A. Clustering
- B. Binary classification
- C. Regression
- D. Object detection

**Correct Answer:** B  
**Explanation:** Binary classification predicts one of two possible outcomes (churn / no churn, yes / no, 0 / 1). Predicting customer churn (will cancel or not) is a classic binary classification problem. Regression predicts a continuous numeric value (e.g., next month's revenue). Clustering groups similar items without predefined labels.

### Q2
**Question:** Which of the following scenarios represents an unsupervised machine learning approach?

**Options:**
- A. Training a model on labeled images of cats and dogs to classify new images
- B. Grouping website visitors into behavioral segments without predefined categories
- C. Predicting house prices based on historical sales data with known prices
- D. Training a model to play chess by rewarding winning moves

**Correct Answer:** B  
**Explanation:** Unsupervised learning finds patterns in data without labeled outcomes. Clustering algorithms (K-Means, DBSCAN) group data points by similarity — the model discovers segments without being told what categories exist. Labeled classification and regression are supervised learning. Rewarding winning moves describes reinforcement learning.

### Q3
**Question:** Which Responsible AI principle states that AI systems should not discriminate against people based on characteristics such as race, gender, or disability?

**Options:**
- A. Reliability
- B. Fairness
- C. Inclusiveness
- D. Privacy

**Correct Answer:** B  
**Explanation:** Microsoft's Responsible AI framework includes six principles: Fairness (AI must treat all people equitably), Reliability & Safety, Privacy & Security, Inclusiveness (AI should empower everyone and engage people), Transparency, and Accountability. Fairness specifically addresses bias and discriminatory outcomes in AI predictions and recommendations.

## Skill Area: Describe fundamental principles of machine learning on Azure (25–30%)

### Q4
**Question:** A data scientist wants to train a machine learning model without writing code, using an interface that automatically tries multiple algorithms and feature engineering approaches to find the best model. Which Azure Machine Learning feature provides this?

**Options:**
- A. Azure ML Designer (drag-and-drop pipeline)
- B. Azure ML Automated ML (AutoML)
- C. Azure ML Notebooks with scikit-learn
- D. Azure AI Studio prompt flow

**Correct Answer:** B  
**Explanation:** Azure Machine Learning Automated ML (AutoML) automatically iterates over multiple algorithms (classification, regression, or time-series forecasting), applies featurization, and selects the best performing model based on a specified primary metric. It requires only a dataset and target column — no code needed for basic usage. Designer uses drag-and-drop components but requires manually choosing algorithms.

### Q5
**Question:** What is the purpose of a "training dataset" and a "test dataset" in machine learning?

**Options:**
- A. The training dataset is used to validate the model's accuracy; the test dataset is used to train the model
- B. The training dataset is used to teach the model patterns; the test dataset is held out to evaluate model performance on unseen data
- C. Both datasets are combined to maximize the training data available
- D. The training dataset contains features; the test dataset contains labels only

**Correct Answer:** B  
**Explanation:** Splitting data into training and test sets is fundamental to building generalizable models. The model learns patterns from the training set. The test set (held out during training) measures whether the model can generalize to new, unseen examples. A model that performs well on training data but poorly on test data is overfitting.

### Q6
**Question:** An Azure ML pipeline is deployed and running daily to score new customer records. Two weeks after deployment, the model's accuracy drops significantly because customer behavior patterns have changed. What is this problem called?

**Options:**
- A. Overfitting
- B. Underfitting
- C. Model drift (data drift or concept drift)
- D. Feature correlation

**Correct Answer:** C  
**Explanation:** Model drift occurs when the statistical properties of the input data (data drift) or the relationship between inputs and outputs (concept drift) change over time, degrading model performance. Azure Machine Learning provides data drift monitoring to detect and alert on these changes, triggering model retraining pipelines.

## Skill Area: Describe features of computer vision workloads on Azure (15–20%)

### Q7
**Question:** A manufacturing company wants to automatically detect defective products on an assembly line by analyzing camera images. They have 500 labeled images of defective and non-defective parts. Which Azure AI service is BEST suited for building this solution with minimal ML expertise?

**Options:**
- A. Azure AI Vision Image Analysis (general-purpose API)
- B. Azure Custom Vision (custom image classification or object detection)
- C. Azure Machine Learning with a custom CNN model
- D. Azure Video Indexer

**Correct Answer:** B  
**Explanation:** Azure Custom Vision allows users to train custom image classification and object detection models by uploading labeled images through a web interface — no deep learning expertise required. With 500 labeled images, Custom Vision can train a production-quality model. The general-purpose Azure AI Vision API doesn't detect domain-specific defects without custom training.

### Q8
**Question:** Which Azure AI Vision capability can extract printed and handwritten text from images and PDF documents, including receipt amounts, form field values, and document text?

**Options:**
- A. Face API — face recognition
- B. Azure AI Vision OCR (Read API)
- C. Azure Video Indexer — transcript extraction
- D. Azure AI Document Intelligence — Layout and Form models

**Correct Answer:** B (also D is valid for forms specifically)  
**Explanation:** Azure AI Vision OCR (the Read API) extracts text from images and PDFs, handling printed and handwritten text in multiple languages. For structured forms (like receipts, invoices, or tax forms) with labeled fields, Azure AI Document Intelligence (formerly Form Recognizer) provides more accurate extraction with field-level structure. The Read API is the general-purpose text extraction solution.

## Skill Area: Describe features of NLP workloads on Azure (15–20%)

### Q9
**Question:** A customer service team wants to automatically categorize incoming support tickets (billing, technical issue, account management, general inquiry) using AI, without manually reviewing each ticket. Which Azure AI Language capability handles this?

**Options:**
- A. Sentiment Analysis
- B. Key Phrase Extraction
- C. Custom Text Classification
- D. Named Entity Recognition

**Correct Answer:** C  
**Explanation:** Custom Text Classification in Azure AI Language trains a model on labeled examples of text mapped to your specific categories. After training, new tickets are classified automatically into the defined categories (billing, technical issue, etc.). Sentiment analysis detects positive/negative/neutral tone. Key phrase extraction identifies important words. Named Entity Recognition detects entities like names, dates, and locations.

### Q10
**Question:** Which Azure AI Language feature can understand a user query like "What is the weather in Seattle tomorrow?" and identify the user's intent ("GetWeather") along with entities like location ("Seattle") and time ("tomorrow") for use in a conversational AI application?

**Options:**
- A. Azure AI Language — Summarization
- B. Conversational Language Understanding (CLU) — intents and entities
- C. Azure OpenAI Service — prompt completion
- D. Question Answering (QnA Maker successor)

**Correct Answer:** B  
**Explanation:** Conversational Language Understanding (CLU) is the successor to LUIS. It extracts intents (what the user wants to do) and entities (key information like location, time, product) from natural language utterances. CLU is the NLU layer for bot and virtual assistant applications, while Question Answering handles FAQ-style question-answer pairs.

## Skill Area: Describe features of generative AI workloads on Azure (15–20%)

### Q11
**Question:** What is a Large Language Model (LLM)?

**Options:**
- A. A database of pre-written text templates for chatbot responses
- B. A deep learning model trained on vast amounts of text data to understand and generate human language
- C. A rule-based natural language processing system
- D. A search engine that retrieves relevant documents based on keywords

**Correct Answer:** B  
**Explanation:** Large Language Models (LLMs) are neural networks with billions of parameters trained on large text corpora using self-supervised learning. They learn statistical patterns of language and can generate coherent text, answer questions, translate languages, summarize content, and write code. Examples include GPT-4o and GPT-4 Turbo available through Azure OpenAI Service.

### Q12
**Question:** A company wants to use Azure OpenAI Service to answer questions about their internal product documentation. They want the model to base its answers ONLY on their documents, not its general training knowledge. What technique is used?

**Options:**
- A. Fine-tuning the base GPT model on the internal documents
- B. Retrieval-Augmented Generation (RAG) — retrieving relevant document chunks and including them in the prompt as grounding context
- C. Increasing the model's temperature parameter to improve accuracy
- D. Using a smaller, specialized model trained only on the product documentation

**Correct Answer:** B  
**Explanation:** Retrieval-Augmented Generation (RAG) retrieves relevant sections of documents from a vector search index (like Azure AI Search) and includes them in the model's prompt as context. The LLM generates answers grounded in the retrieved content. RAG keeps answers accurate and current without the cost and time of fine-tuning. Fine-tuning teaches style/format, not factual knowledge about frequently updated documents.

### Q13
**Question:** Which Azure service hosts OpenAI models (GPT-4o, GPT-4, DALL-E, Whisper) with enterprise security, compliance, and private network access?

**Options:**
- A. Azure AI Studio direct model access
- B. Azure OpenAI Service
- C. Azure Machine Learning model endpoints
- D. Microsoft Copilot Studio

**Correct Answer:** B  
**Explanation:** Azure OpenAI Service is Microsoft's hosted version of OpenAI models, deployed within Azure's secure infrastructure. It provides data residency, private endpoints, VNet integration, content filtering, usage monitoring, and compliance certifications (SOC 2, ISO 27001, etc.). Customer data is NOT used to train OpenAI's models — a key enterprise requirement.
